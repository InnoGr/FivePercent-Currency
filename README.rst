.. title:: Currency

========
Currency
========


Installation
------------

Add **FivePercent/Currency** in your composer.json:

.. code-block:: json

    {
        "require": {
            "fivepercent/currency": "~1.0"
        }
    }


Now tell composer to download the library by running the command:


.. code-block:: bash

    $ php composer.phar update fivepercent/currency
